Local Demo
  1. 配置 ANTHROPIC_BASE_URL 与 ANTHROPIC_API_KEY
  2. 启动 Neo4j：runtime\start_neo4j_harness.ps1
  3. 启动应用：python app.py
  4. 访问 http://127.0.0.1:8010
  5. 如重启 Neo4j，需要重新执行图谱构建
